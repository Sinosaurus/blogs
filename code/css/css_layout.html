此处只要不添加内容没问题，添加了内容就会错位，这是何解 应该是使用 font-size: 0

对左右 div 添加 font-size: 12px 主要用来覆盖父元素的 font-size: 0 ，同时添加了 vertical-align: top 是因为发现盒子对称是按照文字基线来的，加上是行内块，因而考虑如此做 

.item1 {
              font-size: 0;
              /* 清除间距 */
            }
            .item1 .left {
              font-size: 12px;
              vertical-align: top;
              display: inline-block;
            }
            .item1 .right {
              font-size: 12px;
              vertical-align: top;
              display: inline-block;
              width: calc(100% - 100px);
            }
