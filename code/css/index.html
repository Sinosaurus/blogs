行业痛点／需求场景

  • 需求场景

    企业的数据转移是很常见的应用场景，特别是在云计算出现之后，从服务器到云端，从云端到云端的转型需求比比皆是。

  • 行业痛点

    数据迁移有着三个难点：

    首先，大规模的数据需要大量的时间进行传输，人工检查，脚本检测，和反复的扫描，这其中的时间成本和人力成本都是巨大的。

    其次，数据迁移过程中数据本身也在更新，数据的实时同步给迁移的技术提出了很大的要求。

    最后，数据迁移中，内部系统本身仍然需要保持可用，并且需要一直保持使用，但迁移本身会对系统的稳定性造成很大影响，因此对技术也造成了很大的要求。

方案概述

WANDisco的自动化迁移方案在迁移过程中，对数据无需人工确认，且只用扫描一次，节省了大量的迁移成本，并且使用了单通道数据传输，不会影响系统日常的运作，且可以从网络中断中恢复，同时迁移过程中可以实时更新，系统稳定性强，同步性好。在云端，WANDisco接受任意环境的迁移，且可随意拓展节点，云上可拓展性和灵活性非常强。同时，对于计算和存储资源管理能力优异，不浪费资源。

游戏公司：系统上云保证游戏系统实时运行

医疗公司：人事信息上云

政府机关：系统上云保证游戏系统实时运行

金融服务机构：金融交易信息上云，要求实时运行
